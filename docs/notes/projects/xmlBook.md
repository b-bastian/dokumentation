# Fahrtenbuch

Im Fahrtenbuch werden Autofahrten eingetragen. Folgende Daten werden gespeichert:

- Kennzeichen: string
- Startzeit: DateTime
- Endzeit: DateTime
- Startdatum: DateTime
- Enddatum: DateTime
- Start-Km: int
- End-Km: int
- Startpunkt: string
- Endpunkt: string
- Beschreibung: string
